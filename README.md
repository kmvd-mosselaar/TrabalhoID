Este repositório reúne os dois trabalhos desenvolvidos na disciplina de Identidades Digitais, cujo objetivo foi compreender, de forma teórica e prática, como tecnologias criptográficas, blockchain e segurança baseada em hardware podem fortalecer a proteção de identidades digitais.

A disciplina abordou conceitos como:

- Identificadores Descentralizados (DIDs)

- Credenciais Verificáveis (Verifiable Credentials – VCs)

- Atestação remota

- Root of Trust em hardware

- Execução verificável

- Smart Contracts

- Blockchain aplicada à identidade

- Modelos descentralizados de confiança

- Privacidade por design

Aprendizados Desenvolvidos:

- Arquitetura de sistemas de identidade descentralizada

- Integração entre agentes Aries e blockchain

- Desenvolvimento de smart contracts

- Modelagem de confiança distribuída

- Segurança por hardware vs segurança por software

- Análise de trade-offs entre descentralização, custo e privacidade
